Источники правил маршрутизации

← Настройки routing + Добавить источник

Каждый источник хранит список доменов или подсетей с политикой (direct / upstream / block).
Re-import — удаляет старые правила источника и загружает свежие из URL/файла/текста.
Выключить — деактивирует все правила источника в конфиге (без удаления из БД). Включить — активирует обратно.
Удалить — удаляет источник и все его правила из БД безвозвратно.
После любого изменения нажмите ▶ Применить xray-router.
Примеры: …/Russia/inside-raw.lst (домены), …/Russia/inside-cidr.lst (подсети) — itdoginfo/allow-domains.

Подсети: 1.2.3.4, 10.0.0.0/8
Авто: .json → sing-box JSON, .srs → SRS (заглушка), остальное → plain text.
Plain text: пустые строки и строки с # / // игнорируются.
Домены: example.com или .example.com → suffix, full:exact.com → exact, regexp:.*\.ru$ → regex
Подсети: 1.2.3.4 → ip, 10.0.0.0/8 → cidr, cidr:192.168.0.0/16, ip:5.6.7.8
